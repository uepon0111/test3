# サンプルBGM音源の配置方法

イコライザ試聴機能の「サンプルBGM」は、このフォルダに置いた音声ファイルを再生します。
（単純な合成音ではなく、実際に楽器が鳴っている曲でイコライザの効果を確認できるようにするためです）

## 1. 配置するファイル

このフォルダ（`assets/audio/`）に、以下のファイル名で音声ファイルを置いてください。

```
assets/audio/eq-sample-bgm.mp3
```

- 形式: MP3（ブラウザで標準的に再生できる形式です。`.m4a` でも動作しますが、その場合は
  `js/equalizer.js` 内の `fetch('assets/audio/eq-sample-bgm.mp3')` の拡張子も合わせて書き換えてください）
- 長さ: 10秒〜数十秒程度のループ素材がおすすめです（自動でループ再生されます）
- ファイルサイズ: あまり大きいと読み込みに時間がかかるため、1〜3MB程度を目安にしてください

ファイルが無い状態でサンプルBGMを再生しようとすると、「サンプルBGMファイルが見つかりません」という
通知が表示されます（アプリ自体は問題なく動作します）。

## 2. おすすめのフリー音源サイト

商用配布・GitHub Pagesなどでの公開を想定し、**規約上「再配布」や「公開ページへの組み込み」が
許可されているか必ず確認**してから利用してください。サイトによってクレジット表記（曲名・作者名の記載）が
必要な場合があります。

### 日本語サイト
- **DOVA-SYNDROME**（https://dova-s.jp/）— 商用・非商用問わず無料で使える楽曲が豊富。利用規約を要確認。
- **魔王魂**（https://maou.audio/）— BGM・効果音が豊富。クレジット表記の条件等は規約を確認してください。
- **OtoLogic**（https://otologic.jp/）— CC BY 4.0（要クレジット表記）。BGM・効果音が豊富。
- **Music is VFR**（https://dova-s.jp/ とは別サイト／"vfr"で検索）— ジャズ系・おしゃれ系BGMが中心。

### 海外サイト（CC0 / 商用利用可が中心）
- **Pixabay Music**（https://pixabay.com/music/）— 多くがクレジット表記不要のライセンス。検索しやすくおすすめ。
- **YouTube Audio Library**（YouTube Studio内）— Googleアカウントがあれば利用可能。商用利用可の曲が多数。
- **Free Music Archive**（https://freemusicarchive.org/）— CCライセンス曲。曲ごとにライセンス条件が異なるため要確認。
- **Incompetech**（https://incompetech.com/）— Kevin MacLeod氏の楽曲。CC BY（要クレジット表記）。

## 3. 手順まとめ

1. 上記サイトなどから気に入ったBGM・楽器入りの楽曲をダウンロード（MP3形式推奨）
2. ファイル名を `eq-sample-bgm.mp3` にリネーム
3. このフォルダ（`assets/audio/`）に配置
4. GitHub Pagesで公開している場合は、このファイルも忘れずに `git add` → `git commit` → `git push` してください
5. ライセンスでクレジット表記が必要な場合は、設定画面の「このアプリについて」などに曲名・作者名を
   追記しておくと安心です
