# サンプル音源の配置方法

イコライザの「BGMサンプル（外部音源）」は、次のいずれかの名前でこのフォルダに置くと再生できます。

- `equalizer-bgm.mp3`
- `equalizer-bgm.ogg`
- `equalizer-bgm.wav`

推奨は `assets/sample/equalizer-bgm.mp3` です。`mp3` が使えない環境向けに `ogg` を併記しても大丈夫です。アプリは上から順に探して、見つかった最初のファイルを使います。

GitHub に入れるなら、この `assets/sample/` フォルダにそのまま配置してください。コミット後は `https://<your-user>.github.io/<repo>/assets/sample/equalizer-bgm.mp3` のような相対パスで配信されます。

## 取得元のおすすめ

- **DOVA-SYNDROME**: フリーBGM・効果音の定番です。個人利用・商用利用の案内があり、利用条件に従えば背景音楽として使えます。配布ページのダウンロード後ファイルは、サイトから削除されても継続利用できる旨の案内があります。
- **Free Music Archive**: 公式サイトで無料で再生・ダウンロード・共有できると案内されています。収録曲は Creative Commons などのライセンス条件に従う形で提供されます。

曲を選ぶときは、ループしやすい短めの BGM、イントロが静かすぎないもの、楽器数が多すぎないものが、イコライザの差を聴き比べやすいです。
